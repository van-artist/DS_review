查找

查找的概念

查找是指在数据集中根据给定的关键字找到目标元素的位置或内容。

查找算法分类

  1. 静态查找 ：数据集合固定不变，仅查找元素。

  2. 动态查找 ：数据集合动态变化，包含插入和删除操作。

顺序查找

基本思想

从头到尾依次查找目标元素，找到则返回位置，未找到则返回失败。

代码实现：

int SequentialSearch(int arr[], int n, int key) { for (int i = 0; i < n; i++) { if (arr[i] == key) return i; } return -1; // 未找到 }

二分查找

基本思想

对有序数组 ，通过中间元素将查找区间一分为二 ，逐步缩小范围。

代码实现

int BinarySearch(int arr[], int n, int key) { int left = 0, right = n - 1; while (left <= right) { int mid = left + (right - left) / 2; if (arr[mid] == key) return mid; else if (arr[mid] < key) left = mid + 1; else right = mid - 1; } return -1; // 未找到 }

索引结构与B树的应用

索引结构

索引是对数据进行排序并建立指针的一种方式，加快数据查找速度。

B树

B树是一种平衡多路查找树 ，广泛用于数据库索引。

特点：

  1. 每个节点最多有 个子节点（为树的阶）。

  2. 关键字按升序存储，子树节点的值分布在关键字之间。

  3. 所有叶子节点位于同一层。

Hash查找法

基本思想

通过哈希函数将关键字映射到哈希表中的位置，快速定位数据。

冲突解决方法

  1. 开放地址法 ：发生冲突时寻找下一个可用位置。

  2. 拉链法 ：使用链表存储冲突的元素。
